Transparence produit

Ce que Synapse est — et ce qu'il n'est pas encore.

Synapse OMNIA v1.3 est un outil de simulation 3D en accès anticipé (open source, licence MIT). Le dépôt avance vite ; certaines surfaces sont solides, d'autres sont des scaffolds R&D.

Matrice de maturité (honest)

Early Access

Studio & runtime local

  • Synapse Studio (édition, inspecteur, console LLM)
  • 180+ lois physiques dans le catalogue runtime
  • Lois vivantes hot-reload + multiphysique
  • Benchmarks headless reproductibles
  • Import mesh glTF/OBJ, export glTF

Early Access

Rendu & GI L-DNN

  • Pipeline Vulkan + G-DNN / L-DNN fonctionnel
  • Preview procédurale CPU si pas de readback GPU
  • Régression GI automatisée (golden hash CI)
  • Pas de promesse photoréal type Unreal/Lumen

Experimental

Réseau & VR

  • P2P multi-pairs : labo localhost
  • WAN / STUN / TURN : Early Access, pas prod WAN
  • OpenXR : souvent simulé sans casque réel
  • Ne pas planifier de prod collaborative dessus

Benchmarks publics

Mesurez vous-même la couverture du catalogue de lois :

# Combien de lois compilent sans fallback ?
$ dotnet run --project src/Synapse.Studio -- --law-benchmark --law-benchmark-out law-report.json

# Benchmark simulation headless
$ dotnet run --project src/Synapse.Studio -- --benchmark samples/benchmarks/default.json --headless

Dernière mesure interne : ~88 % des lois compilent directement, le reste via fallback de catégorie — 0 échec sur le catalogue complet.

Licence & usage

Synapse est open source (MIT) sur ce dépôt — gratuit pour apprendre, fork, contribuer. Pas de paywall sur le code ; pas de garantie « production-ready » sur les surfaces Experimental.
